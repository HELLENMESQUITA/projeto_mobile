BlackBox

Uma "black box" é um dispositivo, sistema ou processo cujo funcionamento interno é desconhecido ou inacessível aos usuários finais. Este termo é frequentemente usado em várias áreas, como aviação, tecnologia, psicologia e finanças. Na aviação, refere-se a uma caixa que registra dados de voo. Na tecnologia, pode ser um componente com funcionamento oculto. Em psicologia, descreve processos mentais não observáveis diretamente. No desenvolvimento de software, é um tipo de teste sem acesso ao código interno. Nas finanças, refere-se a algoritmos de negociação proprietários. Em resumo, uma "black box" representa um sistema cujo interior é opaco para quem o utiliza.
